    Notice

  • When a static element is given a negative margin on the top/left, it pulls the element in that specified direction
  • When a static element is given a negative margin on the bottom/right,it doesn’t move the element down/right as you might think. Instead, it pulls any succeeding element into the main element, overlapping it.
  • negative margin on right is not presented. Because if an element has another element at its right size, it is usually floated.
